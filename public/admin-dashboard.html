API Key

The Textbelt key is stored server-side as a Firebase secret and is never exposed to the browser. Set or rotate it from a terminal:

firebase functions:secrets:set TEXTBELT_KEY

Then redeploy functions for the new key to take effect.

schedule_send

Automatic sending

When on, the hourly scheduler texts each upcoming service's pastoral-prayer subjects automatically (5 days out, with a 3-day reminder, 8am–8pm Central). When off, only the manual "Send now" button in the Service Builder sends.

These are the texts sent when we ask people for a prayer request before a service. Use {name} where the person's first name should appear (it falls back to "there" if unknown).
